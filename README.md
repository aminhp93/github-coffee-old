# Github Coffee: where developers can ask and connect

Github Coffee is a platform and interactive tool that makes it easy to work with daily task and get only informative news.

## Quickstart (CLI)

- Mac OS X:

  ```
  yarn install
  ```

  ```
  yarn start
  ```
